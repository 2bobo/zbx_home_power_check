#ZabbixでAiSEG2の値を監視するツール
## 概要
AiSEG2から1日の発電量、売電量、買電量、使用量と各回路の使用量を取得してZabbixに送るツール

## 使い方
config.iniの値を環境に合わせて書き換えて使用してください
